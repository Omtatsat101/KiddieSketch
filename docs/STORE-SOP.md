# KiddieSketch Store Operations

## Daily
- Check orders and fulfillment queue
- Review inventory alerts (run `npm run inventory-check`)
- Respond to customer inquiries

## Weekly
- Run SEO audit (`npm run seo-audit`)
- Review analytics (traffic, conversion, top products)
- Update product listings if needed
- Check competitor pricing

## Monthly
- Review and refresh product photos
- Update seasonal collections
- Analyze ad performance
- Plan content calendar

## Theme Updates
1. Pull current theme: `shopify theme pull`
2. Make changes in `theme-customizations/`
3. Test locally: `shopify theme dev`
4. Hand off to Codex for review (if code changes)
5. Push to live: `shopify theme push`
